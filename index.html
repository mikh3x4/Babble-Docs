Babbel Docs

A collaborative editor where everyone reads and writes the same document in their own language. A Google Doc is the storage — one tab per language, translated sentence-by-sentence with Claude.

    Set up for Babbel Docs

    Babbel Docs turns this Google Doc into a multilingual document: it adds a babel:meta tab for settings and keeps one tab per language. Whenever someone edits a sentence in any language, it's retranslated into the other languages with Claude. The current content of the first tab becomes the first language.

    ⚠️ The key is stored inside the Google Doc's babel:meta tab. Anyone with read access to this document can see and use the key. Only share the document with people you trust with it, and use a key with a spend limit.
